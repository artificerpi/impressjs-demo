# impressjs-demo

使用[hovercraft](https://github.com/regebro/hovercraft)和`reStructuredText`制作Slides。

以Coder的方式制作你喜欢的演示文稿

## 预先配置

```bash
sudo pip3 install hovercraft
```

## 启动

```bash
hovercraft index.rst  -p 8080
```

## 快捷键

- H 打开或者关闭提示信息
- 左、下、空格  下一页
- 右、上  上一页
- G   跳到某一页
- P   打开演讲者控制台
